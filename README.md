# 🗄️ Relational Database Engineering Projects

A curated collection of hands-on projects built during the **freeCodeCamp Relational Database Certification**, focusing on **real-world database design, data processing, and backend logic**.

This repository reflects a progression from **fundamentals → practical applications → interactive systems**.

---

## 🚀 About This Repository

Instead of isolated exercises, this repository is structured as a **multi-project portfolio**, where each project explores a different aspect of database engineering.

📌 Each project is located in a separate **Git branch** in this repo.

---

## 📂 Projects Overview

### 🌌 Celestial Bodies Database

Design of a **universe model** with galaxies, stars, planets, and moons.

**Concepts:**

* Relational modeling
* Foreign keys & constraints
* Data organization at scale

---

### ⚽ World Cup Database

Importing and analyzing real match data from a CSV dataset.

**Concepts:**

* Data ingestion (CSV → SQL)
* Joins & aggregations
* Query optimization

---

### 💇 Salon Appointment Scheduler

A CLI-based booking system connected to a database.

**Concepts:**

* User interaction via Bash
* CRUD operations
* Real-world business logic

---

### 🧪 Periodic Table Database

Data cleaning + querying tool for chemical elements.

**Concepts:**

* Database normalization
* Data transformation
* Multi-table joins

---

### 🎯 Number Guessing Game

Interactive game with persistent user statistics.

**Concepts:**

* Stateful applications
* User tracking & analytics
* Backend-driven CLI systems

---

## 🧠 Skills Demonstrated

* Relational Database Design
* SQL (Joins, Aggregations, Constraints)
* Data Normalization
* Bash Scripting for Automation
* CLI Application Development
* Data Cleaning & Transformation
* Git & Version Control

## 🏗️ Repository Structure

Each project is isolated in its own branch:

```bash

```

Example:

```bash
git checkout world-cup-db
git checkout salon-appointment-scheduler
git checkout number_guessing_game
```
